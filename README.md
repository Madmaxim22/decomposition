## О проекте

Учебный проект по **декомпозиции интерфейса** главной страницы Яндекса на независимые React‑компоненты.  
Приложение собрано на **Vite + React + TypeScript** и демонстрирует:

- **Шапку (`Header`)** с логотипом, вкладками, новостями, курсами валют и поиском.
- **Основной контент (`MainContent`)** с промо‑баннером и сеткой информационных карточек.
- **Сетку карточек (`InfoGrid`)** с погодой, картой, телепрограммой и блоком «Эфир».

Компоненты максимально изолированы и переиспользуемы: новости, курсы валют, вкладки, погодный виджет, телепрограмма и трансляции реализованы как отдельные компоненты со строгими пропсами.

## Технологии

- **React 19**
- **TypeScript**
- **Vite**
- **ESLint**

## Структура основных компонентов

- **`App`** – корневой компонент, собирающий шапку и основной контент.
- **`Header`** – верхняя часть страницы: логотип, вкладки, новости, курсы валют, поиск.
- **`MainContent`** – контейнер под шапкой: промо‑баннер и сетка карточек.
- **`PromoBanner`** – широкий баннер под поиском.
- **`InfoGrid`** – сетка информационных блоков:
  - `WeatherCard` – сводка погоды.
  - `MapCard` – блок с картой.
  - `TVScheduleCard` – телепрограмма.
  - `LiveBroadcastCard` – живые трансляции.

## Как запустить локально

Требуется установленный **Node.js** (рекомендуется актуальная LTS‑версия).

```bash
# установка зависимостей
npm install

# запуск режима разработки
npm run dev
```

После запуска откройте адрес, который выведет Vite (обычно `http://localhost:5173/`).

## Сборка и предпросмотр

```bash
# сборка production‑версии
npm run build

# локальный предпросмотр собранной версии
npm run preview
```

## Линтинг
